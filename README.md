# Loss Control App
An angular app for users to fill in a loss control questionaire. 

## Live demo
Live demo is at [loss-control-app.netlify.com](https://loss-control-app.netlify.com/)

## User specifications
- answer questionaire
- review questionaire
- submit questionaire

## Usage
```
ng serve --o
```
